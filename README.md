基于 RAG 的智能问答 Agent 系统
项目概述
本项目基于 Python + LangChain 构建检索增强生成（RAG）智能问答系统，实现本地文档批量入库、向量化存储、语义检索、大模型精准问答全链路能力。项目内置主题知识库素材，支持快速迁移至电商客服、业务知识库、智能咨询等各类场景，适合作为 AI 方向简历项目展示。
技术栈
开发语言：Python 3.9+
核心框架：LangChain
向量数据库：FAISS / Chroma
大模型：兼容 OpenAI 标准接口的大语言模型
文档处理：语义分片、批量向量化、增量更新
工程能力：模块化设计、全链路测试用例覆盖

核心功能
1. 知识库文档管理
支持批量读取本地 TXT/MD 格式文档，自动清洗预处理
自定义语义分片规则，保障长文本语义完整性
支持增量文档追加，无需重建全量向量库
2. 向量检索引擎
将文档内容转化为向量存入本地向量数据库
根据用户问题语义相似度匹配，召回最相关的知识库内容
支持检索结果过滤，减少冗余信息输入，提升回答准确率
3. RAG 智能问答生成
结合检索到的真实文档内容，交由大模型生成回答
约束模型基于知识库内容作答，有效降低大模型幻觉
支持多轮对话上下文记忆，实现连贯的多轮交互
4. 全链路测试体系
根目录提供分层测试脚本，覆盖开发全流程：
最简验证：快速验证核心问答能力
单功能调试：独立测试分片、上传、检索各模块
全链路验证：一键跑通「文档上传→向量存储→检索问答」完整流程
生产工具：带异常捕获、日志记录的正式上传工具
快速开始
1. 安装依赖
bash
运行
pip install langchain chromadb openai python-dotenv faiss-cpu
2. 环境配置
在项目根目录创建 .env 文件（请勿提交到 Git 仓库），填入配置：
env
OPENAI_API_KEY=你的大模型API密钥
VECTOR_DB_PATH=./周鸿林特工/vector_store
3. 运行示例
bash
运行
# 1. 批量导入知识库文档
python test_upload_final.py

# 2. 运行完整问答全流程
python test_full_flow.py

# 3. 最简Demo快速体验
python test_simple.py
